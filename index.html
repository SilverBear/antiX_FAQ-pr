Introdução

Bem-vindo(a) às perguntas frequentes do antiX!

Este documento é destinado ao usuário de primeira viagem olhando para um ambiente de trabalho que difere de um familiar na aparência e na função. Essa diferença é devida aos gerenciadores de janela utilizados para controlar o que o usuário final pode ver e fazer. Esperamos que estas perguntas frequentes lhe dêem uma orientação básica para o antiX e seus gerenciadores de janela e forneçam os meios para explorá-los ainda mais por conta própria.

Configuração Básica Após a Instalação Inicial

  • Como atualizar o sistema? Veja aqui.
  • Como configurar drivers de vídeo nVidia e ATI? Veja aqui.
  • Como autenticar-se automaticamente (autologin)? Ou pela Central de controle -> XWindow -> Autenticação Automática ou manualmente edite o /etc/slim.conf como root, descomente as linhas 82 e 86 e na linha 86 substitua 'demo' pelo seu nome de usuário. Reinicie para verificar.
  • Como editar a lista de fontes do apt?  Ou via Synaptic, Central de controle -> Sistema -> Gerenciar Pacotes -> Configuração -> Repositórios ou editar como root /etc/apt/sources.list
  • Instruções para multimídia? Veja aqui.
  • Como configurar e ativar o Firewall? O FireHol está instalado, mas não habilitado. Abra a Central de controle -> Rede -> Gerenciar Firewall -> Configurar FireHol, mude START_FIREHOL=NO para START_FIREHOL=YES, clique em salvar e em seguida clique em Gerenciar Firewall -> Iniciar FireHol.
  • Como configurar o Wicd para redes sem fio. Veja aqui.

Gerenciadores de Janela

Quais gerenciadores de janela estão disponíveis no antiX?

Dois gerenciadores de janela diferentes vêm instalados e prontos para usar:

  1. o leve IceWM (padrão).
  2. o gerenciador minimalista Fluxbox.

Ambos os gerenciadores de janela podem ser executados com ou sem:

  • o ambiente de trabalho ROX que fornece a funcionalidade de arrastar e soltar para a área de trabalho.
  • o monitor do sistema Conky que exibe informações em tempo real.

Como faço para mudar de um GJ para outro?

No IceWM:

Você pode mudar a partir do IceWM padrão clicando com o botão direito em qualquer lugar do desktop -> Sair... -> Sair da sessão, que o levará de volta para a tela de login do SLiM. Pressione F1 até ver Fluxbox e então autentique-se.

No Fluxbox:

Você pode trocar do Fluxbox para o IceWM com um clique com o botão direito no desktop -> Sair -> Sair da Sessão, o que o levará de volta para a tela de login do SLiM. O padrão é para o IceWM, basta autenticar-se e estará de volta ao IceWM. F1 alterna entre as opções de GJ, além do IceWM e do Fluxbox você não será capaz de autenticar-se em nenhuma outra até que você instale esses gerenciadores de janela ou ambientes de trabalho. Vários podem ser instalador através do metapackage-installer.

E para definir qual é o ambiente padrão?

Central de controle -> XWindow -> Config. Opções de Acesso. Na linha em que se lê "XINITRC_DEFAULT="icewm-session"", substitua "icewm-session" por "startfluxbox" ou por algum outro de sua preferência que esteja instalado.

Ligar/desligar o Conky:

  • IceWM:      clique com o botão direito na área de trabalho ou pressione a tecla Super esquerda para abrir o menu e Área de trabalho -> Configuração -> Conky lig./desl.
  • Fluxbox:    clique com o botão direito na área de trabalho e Área de trabalho -> Área de trabalho -> Conky lig./desl.

Como faço para sair do antiX?

  • IceWM:      na barra de ferramentas você vai ver um ícone de desligar (vermelho) que ao ser clicado oferecerá as opções de bloqueio de tela, hibernação, reinício, saída da sessão, suspensão ou desligamento.
    Também no menu em Sair... há diversas opções, incluindo o desligamento.
  • Fluxbox:    clique com o botão direito em qualquer lugar da área de trabalho e Sair e ser-lhe-ão apresentadas as mesmas opções de acima.

Alguns grandes recursos no antiX.

  • antiX2usb: para aqueles familiarizados com a opção de persistência no Linux Puppy em live-USB, live-CD ou fromiso: bem-vindos! O antiX agora traz um recurso similar chamado antiX2usb que é encontrado na Central de controle -> Discos.
    Você pode usar um sistema live fromiso para instalar o antiX num pendrive com opção de persistência através do antiX2usb e pelo live cd ou pelo sistema instalado você pode fazer o mesmo usando um arquivo iso do antiX ou do MEPIS. Para uma explicação mais detalhada, consulte aqui.
     
  • O metapackage-installer está localizado no menu em Aplicativos -> Acessórios -> metapackage-installer. Você pode instalar pacotes para o Disco, Educação, Jogos, Linguagem, Multimídia, Rede, Não-livre, Escritório, Servidor, Gerenciadores de janela. . . basta escolher o pacote que você deseja e o instalador fará o resto. (conexão de Internet necessária)
     
     
  • O LuckyBackup pode ser encontrado na Central de controle -> Discos -> Backup do Sistema ou no menu em Aplicativos -> Ferramentas -> luckyBackup (super user \)

IceWM: primeiro olhar

O que é isso?

O ambiente de trabalho é executado pelo IceWM, um gerenciador de janelas leve para o X Window System cujo objetivo é a rapidez, simplicidade e não ficar no caminho do usuário.

Onde estão os ícones?

O IceWM não usa ícones mas o antiX traz três aplicativos diferentes que possuem a funcionalidade de gerir ícones na área de trabalho: ROX,  PCManFM e iDesk.

O ROX já vem pré-configurado com alguns ícones no antiX, abra-o em Menu -> Área de trabalho -> Configuração -> Quadro ROX lig./desl.. Para maiores informações sobre o funcionamento do ROX-Desktop, inclusive o gerenciamento de ícones, veja a seção abaixo.

Para usar o PCManFM para isso, abra Menu -> Área de trabalho -> Configuração -> Ícones PCManFM. Ele pode ainda gerir o papel de parede.

Para usar o iDesk para isso, abra Menu -> Área de trabalho -> Configuração -> Ícones IDesk ou a Central de controle -> Área de Trabalho -> Gerenciar Ícones idesk. Isso abre o tidesktool, que é uma interface gráfica para o iDesk e serve para criar os ícones desejados assim como para gerir o papel de parede (veja aqui).

Em todos os três casos acima é preciso habilitar os respectivos programas no arquivo ~/.icewm/startup para que iniciem automaticamente no início da sessão. Abra CC -> Configurar Icewm -> startup e descomente a linha do programa que tiver escolhido, seja o pcmanfm, o idesk ou o rox, abaixo da linha

## Set desktop icons

O que é aquela coisa no canto superior esquerdo?

Essa é a saída do Conky, um monitor do sistema leve que fornece informações em tempo real sobre diversos aspectos do seu computador.

E a barra na parte inferior?

Essa é chamada barra de tarefas ou painel e ela exibe os botões (E -> D) Favoritos, Mostrar área de trabalho, Home, Terminal, Navegador, Volume, Central de controle, Sair, Áreas de trabalho 1-4 (ou desktops) e na extrema-direita, data/hora. Ela é personalizável. Veja aqui.

(Nota: se você passar o ponteiro do mouse sobre os itens na barra de ferramentas você verá os nomes e vamos chamar o botão Favoritos também como Menu.)


IceWM: configuração básica

Como faço para configurar uma conexão sem fio?

Existem 3 opções, todas encontradas na Central de controle do antiX: Wicd, rutilt e Ceni.

1) Na Central de controle -> Rede -> você vai encontrar a interface para rede sem fio Wicd que o antiX usa para fazer a conexão sem fio. Dicas sobre o seu uso podem ser encontradas aqui. Veja também a discussão pormenorizada na Wiki do MEPIS para mais informações e procure no fórum MepisLovers para placas específicas.

2) A ferramenta pela linha de comando (CLI) Ceni dos desenvolvedores do Sidux é um invólucro muito eficiente para a criação de uma conexão: escolha em Hardware interfaces a sua conexão com ou sem fio, clique na sua escolha e você poderá configurá-la em seguida. É um aplicativo bem intuitivo.

3) O Rutilt Wlan Manager também está incluído, localizado na CC antiX e em Aplicativos -> Rede -> rutilt WLAN Manager.

4) O Wpa_gui também está incluído e é encontrado em Aplicativos -> Rede -> wpa_gui.

E quanto à conexão DSL?

Central de controle -> Rede -> Conexão DSL abre a ferramenta de configuração PPPOE. Forneça o nome de usuário e senha para o seu provedor de Internet e opte se deseja ou não que a conexão seja iniciada automaticamente ao carregar o sistema. Para as perguntas que não souber responder opte por 'sim' (o padrão).

E quanto à conexão discada?

Atualmente é bastante difícil encontrar drivers adequados para os antigos e muito comuns softmodems. O site de referência para esses drivers é o Linmodems. No entanto pode ser muito trabalhoso ou até impraticável fazê-los funcionar.

Para aqueles que tenham hardmodems ou tenham encontrado drivers adequados para seus softmodems a Central de controle -> Rede -> Conexão Discada ou Aplicativos -> Rede -> GNOME PPP lançam o Gnome PPP, uma interface gráfica simples para a excelente ferramenta de discagem WvDial.

Como faço para configurar a impressão?

A menos que você tenha uma impressora HP (ver o item seguinte), faça um dos seguintes procedimentos:

1) CC -> Hardware -> Instalar Impressora.  Uma tela será aberta mostrando-lhe as impressoras que foram encontradas, com um ícone para clicar que leva a um assistente para adicionar uma nova impressora.

2) Abra um navegador (Menu -> Navegador), e digite:

http://localhost:631/

Isso levará você para a interface do CUPS, onde você pode configurar sua impressora. Para obter ajuda com impressoras e drivers específicos, verifique a base de dados OpenPrinting.

Como faço para configurar uma impressora HP?

O método mais fácil é usar o metapackage-installer, localizado em Aplicativos -> Acessórios -> metapackage-installer. Abra-o, clique na seta para o Office, marque a caixa de seleção para HP_printing e então, Instalar.

Se você preferir fazer você mesmo, você precisa instalar (ver instalação de software), alguns pacotes adicionais (atenção! Isso pode ser incompleto devido às eventuais subsequentes alterações em pacotes - use por sua própria conta e risco!):

hpijs 
hpijs-ppd 
hplip-data 
hplip 
hplip-GUI 
cups-pdf
cupsys-client 
cupsys-driver-gutenprint 
gs-gpl 
lpr 
magicfilter 
GV 
xprint-utils

Você deve então ser capaz de configurar sua impressora abrindo um navegador (Menu -> Navegador) e entrando em:

http://localhost:631/

Que tal um scanner?

Este deve ser fácil. Conecte o scanner ao computador e clique em Menu -> Aplicativos -> Gráficos -> Xsane Image scanning program.
Você também pode baixar e instalar via Synaptic o Xscanimage. Depois de atualizar o menu, abra-o, selecione entre os scanners que ele tenha encontrado (ele pode ter várias entradas para um scanner, experimente uma a uma até que ela funcione) e você verá uma tela muito básica, cujo funcionamento é bastante evidente. Para ajustar os valores do seu scanner, clique em Menu -> Executar e digite xcam.

Você pode então editar o scan no Geeqie (Menu -> Applicativos -> Gráficos). Para mais detalhes sobre o uso do Geeqie, verifique seu Manual do Usuário.

Qual aplicativo devo usar para a configuração do sistema?

O antiX vem com dois dos Assistentes MEPIS para fazer a configuração do sistema ou o gerenciamento de usuários mais fácil. Eles podem ser encontrados na CC do antiX -> Sistema -> Configurar Sistema ou Gerenciar Usuários.

Detalhes sobre esses utilitários podem ser encontrados na Wiki do MEPIS.

Quero configurar o meu monitor - como posso fazer isso?

O método mais fácil é através da CC -> Hardware -> Resolução de Tela.

Isso irá lançar o aplicativo grandr.

Existem outros Assistentes do MEPIS que eu deva saber?

O Assistente de Usuários do MEPIS reúne uma série de ferramentas de gerenciamento de usuários. É encontrado na Central de controle -> Sistema.

Para mais detalhes sobre esse aplicativo consulte a Wiki do MEPIS.

Posso usar um leitor de MP3 como o iPod com o antiX?

Você pode reproduzir arquivos MP3 com o XMMS e você pode ver e editar etiquetas ID3 instalando o programa EasyTAG. Mas não há nenhuma forma simples para sincronizar seu aparelho com o computador anfitrião sem instalar o Amarok e os arquivos KDE de que ele depende.

Existe uma maneira de interagir com uma câmera digital?

O método mais fácil é simplesmente tratá-la como um dispositivo de armazenamento através de um leitor de cartão USB. Então quando você conectá-la ao computador ela será montada e estará disponível como uma entrada no diretório /media. Você pode clicar em Menu -> Home para examinar e mover ou excluir o conteúdo ou em Menu -> Aplicativos -> Gráficos e optar por abrir o Gtkam ou o UFRaw ou o Geeqie para manipular suas imagens com o programa de sua escolha.

Como faço para instalar o driver da nVidia/ATI no antiX?

A melhor maneira de fazer isso é através do script smxi.

Posso usar uma webcam?

Muitas webcams funcionarão com o antiX, veja a discussão na Wiki do Mepis.

A Cabine de webcam do Cheese está localizada em Aplicativos -> Multimídia -> Cheese.


IceWM: operação básica

Onde estão os meus arquivos?

Como em todas as versões do Linux, os arquivos pessoais estão localizados no antiX por padrão no diretório /home do sistema de arquivos Linux.

Como são geridos os arquivos?

O antiX possui dois gerenciadores de arquivos com interface gráfica. Um deles é o PCManFM, que pode ser aberto clicando no ícone Home da barra de ferramentas ou por Menu -> Home.

O outro é o ROX-Filer, encontrado em Menu -> Ferramentas -> Arquivos -> ROX. Ele é um gerenciador de arquivos que está no coração do ambiente de trabalho ROX (ver a seção ROX). Ele se abre no seu diretório home, onde você poderá mudar a visão para incluir arquivos ocultos do sistema clicando no ícone de olho na barra de menus. Clique com o botão direito do mouse em um arquivo para ver as opções de gestão. Entre elas está Opções..., que gere a configuração do ROX.

Para um gerenciador de arquivos em modo texto, clique em Menu -> Ferramentas -> Sistema de arquivos -> MC, que abre o Midnight Commander, cuja interface principal é composta por dois painéis mostrando o sistema de arquivos com os comandos numerados na parte inferior. Pressione F1 para obter ajuda.

Posso sincronizar arquivos/diretórios no antiX?

Central de controle -> Discos -> Sincronizar Diretórios ou Menu -> Aplicativos -> Ferramentas -> Grsync.

E se eu precisar editar um arquivo de texto?

Você tem cinco boas opções no antiX, desde as mais leves ou simples até as com mais recursos. Duas estão disponíveis clicando em Menu -> Ferramentas -> Editores de texto.

1) O Nano é um aplicativo de linha de comando executado por teclas de controle listadas na parte inferior da tela. Pressione Ctrl-G para acessar o arquivo de ajuda.

2) O Ted foi projetado para tornar possível a edição de documentos tipo rich text em Unix/Linux de um modo WYSIWYG. Arquivos RTF do Ted são totalmente compatíveis com o MS-Word. Além disso, o Ted também é um conversor de RTF para PostScript e de RTF para Acrobat PDF.

3) Para chegar ao Leafpad clique em Menu -> Aplicativos -> Acessórios -> Leafpad. Ele é um aplicativo simples que inclui características essenciais e usa menus familiares.

4) O Geany (Menu -> Aplicativos -> Desenvolvimento -> Geany) possui reconhecimento de linguagens de programação e portanto é adequado ao desenvolvimento de aplicativos.

5) O AbiWord é o processador de texto da suíte de escritório do Gnome. Ele é multiplataforma, compatível com uma grande variedade de formatos de arquivo e extensível por plugins.

Além disso, você pode abrir qualquer texto para edição nos gerentes de arquivos ROX, MC e PCManFM, este último no mesmo local do menu que o Leafpad.

E se um arquivo é compactado em algum formato como .tar etc.?

Clique em Menu -> Aplicativos -> Acessórios -> e abra XArchive Manager, migre para o local do arquivo, selecione o arquivo e clique em Extrair na barra de ícones.

Como faço para instalar e gerenciar software?

Há uma série de métodos de gerenciamento de software no antiX, os mais comuns são a aplicação de linha de comando apt e a sua interface gráfica Synaptic.

  1. Para chegar ao apt, Menu -> Terminal e torne-se root (su seguido por senha de root) ou Menu -> Aplicativos -> Acessórios -> Root Terminal (man apt para instruções completas sobre seu uso). Para a configuração e uso veja a Wiki do MEPIS ou o APT Howto.
  2. Para chegar ao Synaptic, que às vezes é mais fácil para um novato, chame-o com gksu synaptic numa janela de execução ( Menu -> Executar... ou pressionando as teclas Super + barra de espaço). Ele também é encontrado na CC -> Sistema -> Gerenciar Pacotes. Instruções detalhadas podem ser encontradas na Seção 7.2 do Manual do Usuário do MEPIS 8.0.
  3. Para a atualização de todo o sistema, instalação e remoção de certos grupos de aplicativos, instalação de drivers de vídeo e outras coisas mais, pode-se usar o smxi (fundamental se os repositórios do Debian sid estiverem habilitados).

Como atualizo o sistema?

Antes de atualizar o sistema ou instalar novos aplicativos pela primeira vez é necessário atualizar a base de dados dos pacotes. Abra o Synaptic (veja acima) e clique no botão Recarregar ou abra um terminal como root (Menu -> Aplicativos -> Acessórios -> Root Terminal) e
apt-get update
Não utilize o Synaptic para uma atualização do sistema, para isso será preciso usar o terminal, seja com o APT, o Aptitude ou o smxi. O antiX é baseado no MEPIS que por sua vez é baseado no Debian, mas diferentemente do MEPIS, o antiX é construído sobre os repositórios do Debian 'testing'. Portanto para atualizar o sistema, num terminal como root:
apt-get dist-upgrade
Se houverem mensagens de erro durante a atualização, veja se encontra a solução nos fóruns do antiX em http://antix.freeforums.org ou em Mepislovers. Se necessário, abra um novo tópico para esclarecer a sua dúvida.

Para informações sobre o uso e recursos do APT, digite num terminal man apt. Aproveite e explore também man apt-get, man apt-cache, man sources.list etc.

Depois de instalar novos programas como faço para encontrá-los e como faço para pô-los no menu?

  1. Menu -> Área de trabalho -> IceWM -> Atualizar menu (auto) irá automaticamente atualizar o seu menu e colocar nele os novos programas.
  2. Menu -> Área de trabalho -> IceWM -> Editar menu (manual) abrirá o programa IceMC, um editor de menus gráfico (em inglês) para o IceWM, projetado para ser simples e estável. Clique em Open File e abra menu. Você pode configurar as entradas de menu com copiar, colar e arrastar e soltar. Familiarizar-se com a organização do arquivo ~/.icewm/menu facilita a compreensão e o uso do IceMC.

Como posso obter o meu e-mail?

Abra Menu -> Aplicativos -> Rede -> Iceape Mail & Newsgroups. Você entra no setup, seguindo o assistente que aparece a primeira vez que utilizar o aplicativo ou a qualquer momento ao clicar na barra de menus em Configuration -> Create new account... e preenchendo os espaços em branco com as informações de sua conta de e-mail.

O que o antiX oferece para navegar na Internet?

O navegador padrão é o completo Iceape, a versão do Debian para o Seamonkey livre de marcas registradas. Clique em Menu -> Navegador ou no ícone da barra de ferramentas para lançá-lo. Ele é compatível com muitas extensões feitas para o Firefox. Veja aqui.

Dois outros navegadores leves também estão incluídos em Menu -> Aplicativos -> Rede. O Dillo é um rápido e minimalista navegador multiplataforma que é altamente seguro, enquanto o navegador Links2 é um navegador web modo texto e gráfico que também é muito rápido e seguro e tem menus disponíveis clicando-se na barra superior.

Também pode-se lançar o script exoodles como root e ele oferece, entre outras opções de multimídia, a instalação dos navegadores Google Chrome, Opera e Swiftfox.

Posso bater papo com o antiX?

Existem três programas de chat que vêm instalados no antiX, os dois primeiros estão localizados em Menu -> Aplicativos -> Rede

  1. O Chatzilla é um programa de IRC simples. Preencha as informações de conexão e você está pronto para papear. Se precisar de ajuda, consulte as FAQ.
  2. O Pidgin (antes chamado Gaim) inicia um cliente de mensagens instantâneas multiprotocolo que lhe permite usar todas as suas contas de mensageiros ao mesmo tempo. A configuração é intuitiva e se precisar de ajuda pode verificar estas perguntas mais frequentes.
  3. Finalmente, clicando em Menu -> CLI -> Rede -> Bate-papo abre-se um rápido e muito capaz cliente de IRC em modo texto. A configuração e o uso não são óbvios, portanto não deixe de verificar a documentação do Irssi.

O que posso usar para gravar um CD/DVD?

Para uma interface gráfica clique em Menu -> Aplicativos -> Multimídia -> GnomeBaker CD/DVD Writer.
Para um aplicativo em modo texto Menu -> CLI -> Multimídia -> Queimador de cd -cli irá executar o cdw.

O que eu uso como um agregador de notícias (RSS)?

Você tem algumas boas opções:

  1. Use um agregador online como o Bloglines que você acessa através de um navegador.
  2. Use Menu -> Aplicativos -> Rede -> Iceape Mail & Newsgroups. É bastante intuitivo de usar.
  3. Instalar um agregador de desktop a partir dos repositórios (use o Synaptic e pesquise sobre alimentação de RSS).
  4. Menu -> CLI -> Rede -> Leitor de RSS/Atom lança o aplicativo CLI Newsbeuter.

Como faço para tocar música?

Para reproduzir um CD de áudio ponha-no em seu CD player e (não há autoplay) clique em Menu -> Aplicativos -> Multimídia - > xmms. Isso abre o XMMS, um player multimídia para sistemas Unix. Clique no botão avançar (segundo da esquerda) para iniciar o CD. Na primeira vez será necessário navegar até  diretório /media/cdrom (nota: ../ significa o diretório superior) e adicionar as faixas que se deseja ouvir.

Você pode selecionar uma fonte diferente de arquivos de áudio para tocar no XMMS. Para fazer isso, dê um clique com o botão direito na barra superior do player XMMS e do menu pop-up selecione Play File para selecionar o arquivo de áudio para tocar. Para reproduzir um áudio MP3 ou arquivo Ogg Vorbis simplesmente clique sobre ele e o XMMS abre automaticamente e começa a tocar o arquivo.

Há também um mixer disponível na Central de controle -> Hardware -> Ajustar Misturador ou clicando no ícone Volume na barra de tarefas e em seguida no ícone MIXER. A aplicação é o AlsaMixer.

Se você quiser um player no terminal para as suas músicas no disco rígido, abra Menu -> CLI -> Multimídia -> Cliente de música - moc.

E sobre a reprodução de música de uma fonte na Internet?

Se você quiser reproduzir um arquivo de áudio da Internet, abra o XMMS, escolha Play Location e informe o endereço do arquivo na Internet (URL).

Alternativamente você pode usar o Menu -> Aplicativos -> Multimídia -> streamtuner, que oferece centenas de milhares de músicas em uma interface familliar fácil e limpa.

Um gerente de músicas muito bom também está instalado: Goggles Music Manager.
Citando o site do programa, "O Goggles Music Manager é um gerenciador de coleções de música e tocador que automaticamente classifica arquivos Ogg Vorbis, FLAC, MP3, MP4, ASF e Musepack com base em gênero, artista, álbum e canção. Ele suporta reprodução contínua (sem silêncios), fácil edição de etiquetas e permite scrobbling (log automático) de faixas para o last-fm."

Posso extrair música com o antiX?

Para um aplicativo CLI, Menu -> CLI -> Multimídia -> Ripador lançará RipIT v3.8.0.

Para uma GUI, Menu -> Aplicativos -> Multimídia -> Asunder CD Ripper. Esse aplicativo pode gravar faixas de um CD de áudio como WAV, MP3, OGG, FLAC e/ou WavPack.

Há também outro aplicativo CLI no antiX chamado abcde.

O DVD do filme que eu coloquei não funcionou. Como posso assistir aos filmes?

Não há recurso de execução automática, então após colocar o seu DVD ele poderá ser reproduzido usando o gxine, o xine, o GNOME MPlayer ou o MPlayer.

Menu -> Aplicativos -> Multimídia -> GNOME MPlayer: navegue por Arquivo -> Disco para iniciar a reprodução.

Menu -> Aplicativos -> Multimídia -> MPlayer Media Player: clique com o botão direito sobre o programa aberto e em Open... -> Play DVD.

Outras opções são Menu -> Aplicativos -> Multimídia -> gxine ou xine. Eles procuram pelo DVD em /dev/dvd. Como o antiX pode identificar o seu gravador de DVD por um outro dispositivo, verifique se /dev/dvd existe e se não existir crie uma ligação simbólica do seu dispositivo de DVD para /dev/dvd (use o gerente de arquivos ROX como root para isso).

Também pode ser útil verificar para qual região o DVD está configurado. No gxine: File -> Configure -> Preferences... -> media -> dvd. No xine clique com o botão direito na tela e Settings -> Setup... -> media.

O MPlayer também pode ser usado para a reprodução de DVDs ao ser lançado pela linha de comando (ver man mplayer para mais informações).

Para reproduzir um arquivo de vídeo (por exemplo, AVI, MPEG etc.), simplesmente clique sobre ele e o GNOME mplayer ou o MPlayer será aberto automaticamente e iniciará a reprodução do arquivo. Quando aberto com o MPlayer, use a tecla p ou de espaço para pausar o filme, qualquer tecla para reiniciá-lo e q ou Esc para parar a reprodução e fechar o programa (mais detalhes pelo comando mplayer --help).

Que aplicativos o antiX oferece na área de criação?

Alguns dos que se encaixam nessa classificação são o criador de apresentações de slides Imagination (no submenu Multimídia), o editor de áudio mhWaveEdit (Multimídia), o seletor de cores Gcolor2 (Gráficos), o conversor de imagens RAW UFRaw (Gráficos), o visualizador e manipulador de imagens Geeqie (Gráficos) e o editor de imagens mtPaint (Gráficos).

Existe um protetor de tela?

Sim, o padrão é uma tela preta, com ou sem bloqueio. Alternativamente use a opção de bloqueio pelo ícone Sair ou por Menu -> Sair... -> Bloquear estação. Se você quiser, pode adicionar o xscreensaver pelo Synaptic ou, naturalmente, pela linha de comando.

Como faço para manipular uma janela aberta?

Além das operações padrão com os três botões no canto superior direito (em alguns temas acrescidos pelo enrolar/desenrolar), outras possibilidades (ícone na bandeja, camadas etc.) estão disponíveis através do botão direito do mouse na barra de título. Um clique duplo na barra de título restaura/maximiza a janela. Você também pode redimensioná-la arrastando o canto inferior direito.

Quais aplicativos estão disponíveis para uso em escritório?

Todos os aplicativos de escritório são encontradas sob Menu -> Aplicativos -> Escritório. O antiX vem com a planilha leve Gnumeric e um processador de textos também relativamente leve chamado AbiWord. Ambos são capazes de abrir arquivos do MS-Office. Para uma maior funcionalidade, instale o openoffice.org dos repositórios após certificar-se de que há espaço suficiente no disco rígido.

Outros aplicativos de escritório incluem um leitor de PDF, uma calculadora e um gerenciador de informações pessoais chamado Osmo.


IceWM: personalização básica

Como se personaliza o IceWM?

É feito editando diretamente os arquivos de configuração. Para isso abra Central de controle -> Área de Trabalho -> Configurar IceWM.

Para alterar as preferências do IceWM vá para Menu -> Área de trabalho -> IceWM -> Central de controle IceWM. Lá você pode ajustar a barra de tarefas, applets, áreas de trabalho e janelas. Nota: para editar o menu, veja aqui.

O que eu posso mudar?

Você pode alterar o que quiser, seguindo as orientações do Manual do IceWm.

Quais são as mudanças mais comuns que se faz no painel?

Vamos nos concentrar em quatro alterações comuns para o que você vê e como ele funciona. Abra o gerente de arquivos ROX como descrito anteriormente, clique no olho para exibir os arquivos ocultos, clique em .icewm e, finalmente, em preferences para abrir esse arquivo em um editor de texto. Quando você terminar com as alterações, salve e saia. Elas terão efeito na próxima vez que você se autenticar no IceWM ou reiniciá-lo: Menu -> Sair... -> Reiniciar IceWM.

1) Mudar para um relógio de 12 horas: procure (Ctrl+F):

timeformat

e deixe assim:

TimeFormat="% I:% M"

Note que a linha foi descomentada e o texto entre aspas foi alterado.

2) Variação do número de espaços de trabalho: procure (Ctrl+F):

workspacenames

Altere a entrada como preferir; por exemplo, pode-se ler:

WorkspaceNames=" 1 "," 2 "

3) Fazer com que o ícone do correio funcione: procure (Ctrl+F):

mailbox

Altere a entrada abaixo para:

MailCommand="seu programa de correio"

Novamente, observe que a linha foi descomentada e o texto entre aspas foi alterado.

4) Habilitar a ocultação automática do painel: procure (Ctrl+F):

autohide

Altere a linha abaixo para ler:

TaskBarAutoHide=1 # 0 / 1

Note que a linha foi descomentada e o valor alterado para "1" (verdadeiro).

Como faço para mudar o menu e a barra de ferramentas?

Você pode fazê-lo da mesma forma que com preferences, editando os arquivos menu e toolbar ou você pode usar o aplicativo IceMC via Menu -> Área de trabalho -> IceWM -> Editar menu (manual).

E quanto aos atalhos de teclado?

Central de controle -> Área de trabalho -> Configurar IceWM -> keys. A sintaxe é bem simples. "Alt+Ctrl" equivale à tecla "Super".

Como faço para mudar o fundo de tela?

Via Central de controle -> Área de Trabalho -> Escolher Tema de Fundo. Observação: você pode definir uma imagem diferente para cada GJ!

Você ainda pode usar para isso os mesmos aplicativos que usa para gerir os ícones na área de trabalho: ROX, PCManFM ou iDesk.

Para configurar o fundo de tela com o ROX, veja aqui.

Para o PCManFM, abra Menu -> Área de trabalho -> Configuração -> Ícones PCManFM e marque a caixa Papel de parede.

Para usar o iDesk, abra-o em Menu -> Área de trabalho -> Configuração -> Ícones IDesk ou pela CC -> Área de Trabalho -> Gerenciar Ícones idesk, clique em preferências e adicione as linhas abaixo após a linha 'table Config':

  Background.File: image.jpg
  Background.Delay: 1
  Background.Source: ~/Wallpaper
  Background.Mode: Scaled
  Background.Color: #C2CCFF

As configurações acima farão com que as imagens em ~/Wallpaper se alternem a cada 1 minuto como fundo de tela. Para mudar a periodicidade (em minutos), alterne o valor de 'Background.Delay:'. Se você quiser apenas uma imagem fixa, ponha o seu caminho na linha 'Background.File:' e deixe 'Background.Source:' vazio. Para mais informações, veja aqui.

Se você optar por usar o ROX, o PCManFM ou o iDesk para o gerenciamento do fundo de tela ao invés do programa padrão no antiX que é o Nitrogen, descomente as suas respectivas linhas em ~/.icewm/startup (no exemplo abaixo estamos habilitando o iDesk)

## Set desktop icons
#pcmanfm -d &
idesk &
#rox --pinboard=antiX &

e comente as linhas do nitrogen

## Set wallpaper via nitrogen
#rm $HOME/.config/nitrogen/bg-saved.cfg
#ln -s $HOME/.config/nitrogen/bg-saved-icewm.cfg $HOME/.config/nitrogen/bg-saved.cfg 
#nitrogen --restore &

Se você usar o ROX ou o PCManFM, será preciso editar ~/.conkyrc (CC -> Sistema -> Editar Conky) se quiser visualizar o Conky. Mude 'own_window no' para 'own_window yes'.

Como faço para desativar o Conky?

Clique em Menu -> Área de trabalho -> Configuração -> Conky lig./desl.

E se eu precisar editar um arquivo de texto?

Você tem cinco boas opções no antiX, desde as mais leves ou simples até as com mais recursos. Duas estão disponíveis clicando em Menu -> Sistema -> Editores de texto.

1) O Nano é um aplicativo de linha de comando executado por teclas de controle listadas na parte inferior da tela. Pressione Ctrl-G para acessar o arquivo de ajuda.

2) O Ted foi projetado para tornar possível a edição de documentos tipo rich text em Unix/Linux de um modo WYSIWYG. Arquivos RTF do Ted são totalmente compatíveis com o MS-Word. Além disso, o Ted também é um conversor de RTF para PostScript e de RTF para Acrobat PDF.

3) Para chegar ao Leafpad clique em Menu -> Aplicativos -> Acessórios -> Leafpad. Ele é um aplicativo simples que inclui características essenciais e usa menus familiares.

4) O Geany (Menu -> Aplicativos -> Desenvolvimento -> Geany) possui reconhecimento de linguagens de programação e portanto é adequado ao desenvolvimento de aplicativos.

5) O AbiWord é o processador de texto da suíte de escritório do Gnome. Ele é multiplataforma, compatível com uma grande variedade de formatos de arquivo e extensível por plugins.

Além disso, você pode abrir qualquer texto para edição nos gerentes de arquivos ROX, MC e PCManFM, este último no mesmo local do menu que o Leafpad.

ROX: primeiro olhar

O que é este ambiente de trabalho?

Esse é o ROX Desktop, baseado em torno do sistema de arquivos Linux. Seu componente principal é ROX-Filer, um gerenciador de arquivos poderoso. Outros componentes disponíveis no antiX são o painel ROX-Panel e o quadro ROX-Pinboard, que podem ser habilitados separadamente do Fluxbox no submenu Área de trabalho -> Área de trabalho e no IceWM no submenu Área de trabalho -> Configuração.

O que se pode fazer com o ROX-Filer?

Bem, gerenciar arquivos, é claro! Mas ele tem muitos outros recursos que são cobertos abaixo e nos documentos listados sob Links. O manual do gerente de arquivos ROX também está disponível off-line em /usr/share/doc/rox-filer/html/ ou pelo Exibir a ajuda do ROX-Filer no seu menu.

Para que serve o ROX-Desktop?

Sua grande vantagem é permitir a você usar ícones para qualquer item no sistema de arquivos com um simples arrastar e soltar. Nota: se habilitado o quadro ROX, a configuração do Conky deve ser alterada para `own_window yes' e `own_window_type desktop'.

E o ROX-Panel?

Sua principal função é permitir-lhe criar e personalizar o seu próprio painel.


ROX: configuração básica

Como posso evitar ter de lançar o ROX-Desktop toda vez?

O ROX-Desktop pode iniciar automaticamente através de uma simples edição no Fluxbox ou no IceWM. Em Central de controle -> Área de Trabalho -> Configurar Fluxbox/IceWM -> Startup, remova o # dessa linha: 

# rox --pinboard=antiX &

Salve e saia/autentique-se na sessão para ter seu ROX-Desktop pré-personalizado pelo antiX (veja abaixo) utilizando o gerenciador de janelas de sua escolha. Restaure o símbolo de comentário à linha acima se você quiser acabar com o uso permanente do ROX-Desktop.

Se quiser iniciar também o painel, adicione a seguinte linha ao arquivo startup do IceWM/Fluxbox:

rox -t=meupainel &

A opção -t (top) põe o painel na margem superior, -l (left) põe-no à esquerda, -r (right) à direita e -B (bottom) embaixo (note o 'B' maiúsculo). Mais informações aqui. rox -h num terminal mostra todas as opções e os seus significados.

Nota: se você optar por ter o ROX-Desktop iniciando automaticamente então as opções painel e quadro no submenu do Fluxbox Área de trabalho -> Área de trabalho ou no submenu do IceWM Área de trabalho -> Configuração não devem mais ser usadas.


ROX: operação básica

Quais são os movimentos básicos para usar com o ROX-Filer?

  • Botão esquerdo do mouse sobre o arquivo/pasta para abrir seu conteúdo.
  • Botão do meio em uma pasta abre ela em uma janela separada.
  • Botão do meio em um arquivo abre o arquivo e fechar a janela do gerenciador de arquivos.
  • Botão direito do mouse em um arquivo/pasta abre a caixa de diálogo do ROX, onde você pode ver e definir opções.

Estas são as opções padrão e podem ser modificadas clicando-se com o botão direito no fundo de uma janela do ROX e escolhendo Opções....

Como faço para mover arquivos?

Isso é provavelmente mais fácil com duas janelas abertas: basta arrastar de uma para a outra.

Como faço para colocar ícones no meu desktop ROX?

É muito fácil! Para adicionar ícones de atalho a dispositivos, aplicações, pastas e arquivos no desktop basta arrastar e soltar a partir do ROX-Filer.

Que tal um dispositivo, por exemplo um CD-ROM ou drive de disquete?

Abra o ROX-Filer e navegue até a pasta /media. Clique sobre o dispositivo que você desejar, arraste e solte para onde você quiser na tela.

Posso criar atalhos para aplicações?

Você pode fazer isso também, mas primeiro precisa saber onde se encontra o executável do programa. Geralmente está em /usr/bin ou /usr/local/bin mas você pode encontrá-lo abrindo um terminal e digitando:

which NomeDoAplicativo

Depois de encontrá-lo abra o ROX-Filer e navegue até a sua localização. Clique sobre o arquivo do aplicativo (se houver uma ligação simbólica, escolha esta) e - aqui vamos nós outra vez - arraste e solte para onde quiser na tela.

Com diretórios e arquivos faço da mesma maneira?

Exato.

Quais são os truques com o ROX-Panel?

Você pode usar as mesmas técnicas descritas acima, só que arraste até o painel em vez de ao desktop. Para remover, clique com o botão direito do mouse e selecione Remover Item(ns).

Eu ainda posso ver o painel do Fluxbox apesar de eu ter alternado para o ROX-Panel. O que devo fazer?

Ele desaparece assim que o cursor está sobre o ROX-Panel, mas há uma gambiarra fácil: clique com o botão direito do mouse no painel do Fluxbox e selecione Esconder Automaticamente para que ele desapareça quando não estiver em uso.


ROX: personalização básica

Como faço para alterar o papel de parede no quadro ROX?

Um método é:
1) Clique na pasta home, navegue até ~/Rox e clique em Wallpaper.
2) Clique em Imagens azulejadas, Imagens escalonadas, Imagens esticadas ou Imagens centralizadas (qual for mais adequado à imagem a ser utilizada).
3) Abra outra janela do ROX-Filer, navegue até onde se encontra a imagem que deseja usar e arraste-a para o diretório Imagens azulejadas/escalonadas/esticadas/centralizadas (o que você abriu).
4) Se você quiser que o fundo se alterne de tempos em tempos, arraste mais alguma(s) imagem(ns) para lá e defina Mudar a cada x minuto(s) conforme preferir. O Wallpaper utilizará as imagens dos quatro diretórios acima.
5) Clique em Ativar o modo Aleatório e em seguida Fechar.

Outro método é:
1) Coloque o seu papel de parede na pasta Wallpaper no seu diretório home.
2) Clique na pasta home, navegue até Wallpaper e arraste e solte a imagem escolhida para o desktop.
3) Clique com o botão direito do mouse sobre o arquivo arrastado e escolha Fundo..., em seguida arraste o arquivo para o espaço onde se diz Arraste uma imagem para cá. Selecione o tratamento da imagem no alto (Centralizar, Escalonar etc.)
4) Clique com o botão direito do mouse sobre o arquivo de imagem arrastado e selecione Remover Item(ns).

Mais informações podem ser encontradas no guia de personalização do desktop.


ROX: avançado

Como posso definir a aplicação padrão para abrir um arquivo?

Se desejar alterar a aplicação padrão que será usada para abrir um tipo de arquivo, siga estas instruções:
1) Clique com o botão direito do mouse sobre o arquivo que você está interessado em mudar e clique em Arquivo -> Definir Ação Para Execução...
2) Digite o nome do aplicativo na caixa denominada Informe um comando shell. Observação: deixe um espaço entre a última letra do aplicativo e o sinal ".
3) Salve e o arquivo agora será aberto no aplicativo de sua escolha.

É possível ter mais de uma configuração para o quadro?

É sim. Os scripts startup do IceWM e do Fluxbox abrem um quadro personalizado (chamado 'antiX') e Menu -> Área de trabalho -> Área de trabalho -> Quadro ROX lig./desl. (Fluxbox) e Menu -> Área de trabalho -> Configuração -> Quadro ROX lig./desl. (IceWM) abrem o quadro padrão (chamado 'Pinboard') com uma personalização mínima (a imagem de fundo). Os seus arquivos de configuração encontram-se em ~/.config/rox.sourceforge.net/ROX-filer/ sob os nomes de pb_antiX e pb_Pinboard respectivamente. Se você abre o quadro pelo menu e faz as mudanças que desejar, elas serão salvas em pb_Pinboard e retornarão quando você abrir o quadro novamente. Se você quiser criar um outro quadro com uma configuração diferente abra Menu -> Executar e digite

rox -p=meuquadro
(substitua 'meuquadro' pelo nome que quiser). Isso abrirá um quadro padrão; modifique-o à vontade e ele será automaticamente salvo sob o nome dado. Então você poderá adicionar uma entrada para ele no menu do Fluxbox/IceWM (ver abaixo). Note que -p é o mesmo que --pinboard (digite rox -h num terminal para conhecer todas as opções e seus significados).

Posso fazer com que o quadro ROX abra o meu próprio ao invés de um padrão?

Você pode lançar o seu quadro pessoal editando o menu do Fluxbox como descrito anteriormente. Abra ~/.fluxbox/menu num editor de texto e adicione essa linha num lugar apropriado:

[exec] (Meu quadro) {rox --pinboard=meuquadro}

Pra o menu do IceWM (~/.icewm/menu), seria

prog "Meu quadro" /caminho/ao/ícone/ou/um/traço/-/para/nada rox --pinboard=meuquadro

Certifique-se de substituir 'meuquadro' pelo nome real do quadro que você deseja carregar. Dessa forma vários quadros podem ser configurados e relacionados no menu do Fluxbox/IceWM. Muito conveniente!

Posso personalizar os ícones?

Se você tiver adicionado ícones no desktop, então:
1) Clique com o botão direito do mouse sobre o tipo de ícone que você deseja alterar e em Arquivo (ou Dir) -> Definir o Ícone...
2) Selecione uma das opções disponíveis (somente para o arquivo/diretório, para todos do mesmo tipo etc.)
3) Agora arraste e solte na caixa o ícone de outra janela do ROX e voilà! O ícone escolhido agora aparece no desktop (se definido para todos do mesmo tipo é necessário reiniciar - fechar/abrir - o quadro ROX).

Note que ao escolher um tema de ícones para o ROX-Filer ele se estende aos itens no desktop (menos para os itens que tenham sido modificados individualmente). Para isso clique com o botão direito no fundo de uma janela do gerente de arquivos ROX -> Opções... -> Tipo -> Tema de ícones, escolha um entre os disponíveis e clique OK (aplica-se imediatamente para o gerente de arquivos mas é necessário reiniciar o quadro para que as mudanças se façam no desktop).

Como faço para editar o nome de um atalho?

Clique com o botão direito do mouse sobre o ícone, em seguida em Editar o Item e altere o nome exibido na caixa O texto sob o ícone é:. Clique OK para finalizar.

O que são as teclas de atalho do ROX?

As teclas de atalho no ROX são uma combinação específica de teclas para abrir uma pasta ou arquivos com um único ou no máximo 2 cliques. Veja a explicação detalhada.

Conky: primeiro olhar

O que é essa coisa que eu vejo no canto superior esquerdo?

Para que serve?

O Conky pode exibir qualquer coisa que você queira monitorar, seja sobre a área de trabalho ou em sua própria janela.

Você pode me dar alguns exemplos?

Claro: espaço em disco, carga do sistema, status da rede sem fios, estado da bateria, velocidade de download e de upload, conexões de Internet por protocolo, uso da RAM, utilização de swap, os processos em execução, previsão do tempo etc.


Conky: configuração básica

Como faço para configurá-lo?

O Conky já vem instalado e configurado por padrão no antiX para exibir informações básicas do sistema. Abra ~/.conkyrc num editor de textos ou a Central de controle -> Sistema -> Editar Conky e dê uma olhada nas informações do arquivo. Você verá que ele traz comentadas inclusive algumas opções já configuradas para laptops (como monitoramento do estado da bateria). Se quiser exibi-las, descomente-as (remova o sinal # do início da linha) e salve.

Conky: personalização básica

Como posso mudar a aparência do Conky ou as informações exibidas?

Siga esses passos:
1) Abra a Central de controle -> Sistema -> Editar Conky. Isso abrirá um editor de texto com o seu script .conkyrc.
2) Antes de começar a editar, faça uma cópia de seu script padrão do Conky (~/.conkyrc) com um nome diferente como "conkyrc_original". Dessa forma se você cometer um erro ou não gostar das alterações, você pode recarregar o script padrão.
3) Faça as alterações que você gostaria na configuração do Conky, salve e saia. Talvez você precise clicar duas vezes em Menu -> Área de trabalho -> Configuração -> Conky lig./desl. para desativá-lo e reativá-lo para ver as alterações. A seção de opções (options) no arquivo ~/.conkyrc fornece algumas dicas úteis.

Posso usar o terminal para fazer isto?

Você pode editar o arquivo de configuração do Conky em um terminal, digitando (substitua pelo seu nome de usuário):

nano /home/usuário/.conkyrc

O que é o smxi?

O smxi é um script escrito por um programador chamado h2 que permite aos usuários manter o seu antiX atualizado constantemente.

O que pode fazer o smxi?

Quando executado, o smxi permite aos usuários instalar ou remover um kernel, instalar os drivers gráficos da ATI e da nVidia, executar apt-get dist-upgrade com segurança, fazer do antiX um sistema baseado no Debian sid ou no sidux e muito mais!

O smxi é auto-explicativo. Para ver as opções de uso, execute smxi -h.

Como se usa o smxi?

O smxi já está instalado no antiX-M8.5 e pronto para ser executado. O smxi deve ser executado fora do X Window System. Para sair do X, quando em seu desktop, pressione Ctrl-Alt-F1 para chegar a um terminal, autentique-se como root e digite: 

smxi

O smxi então vai interromper o X e fará uma série de perguntas na primeira vez que é executado, incluindo opções de qual sistema você deseja executar (isto é, padrão, sidux ou sid). anticapitalista, o desenvolvedor do antiX, sugere selecionar a opção de não-mudança continue-no-changes, usando o apt-get (ao invés de aptitude) e apt-get dist-upgrade.

Para executar o smxi antes de carregar a área de trabalho, ponha um 3 no fim da linha kernel da entrada do antiX no menu do GRUB (pressione e durante a tela do menu para poder editá-lo). Assim o sistema carregar-se-á para uma tela de terminal. Após a sua execução o smxi oferecerá a opção de carregar a área de trabalho.

E o sgfxi? O que ele faz?

O sgfxi é o script do smxi que lida com a instalação de drivers gráficos. Como o smxi, ele deve ser executado fora do ambiente de trabalho X. Logado como root, entre: 

sgfxi

O que é o inxi?

Também está incluído no antiX o inxi, um script do H2, o autor do smxi e do sgfxi. O inxi é um script de linha de comando que fornece informações do sistema. Execute inxi -h para todas as opções disponíveis, para atualizar execute inxi -U.
Se você quiser mais informações visite o fórum de usuários  do smxi/sgfxi/inxi (em inglês).
